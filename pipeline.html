Cyclical Time Features

  • Hour: sin/cos encoding (24-hour cycle)
  • Day of week: sin/cos encoding (7-day cycle)
  • Month: sin/cos encoding (12-month cycle)

  • Handles non-stationary trends and seasonal variations
  • Prevents data leakage from future periods

Stage 3: Sliding Window Dataset Creation

Window Configuration

  • Look-back window (W): 48 hours = 576 steps
  • Forecast horizon (H): 48 hours = 576 steps
  • Input shape: [batch, 576, n_features]
  • Target shape: [batch, 576, 1] (TOTAL only)

Data Splits

  • Training: Until 2023-12-25
  • Validation: 2023-12-26 to 2024-11-25
  • Test: After 2024-11-25
  • Time-aware splitting prevents data leakage

Encoder Components

  • Two-layer LSTM encoder (256‚Üí128 units)
  • LayerNorm for training stability
  • Attention mechanism for temporal focus
  • Compresses sequence to fixed-size latent vector

Forecasting Strategy

  • Concatenates latent vector + last observed features
  • Multi-layer forecasting head with dropout
  • Direct multi-step prediction (no autoregression)
  • Outputs full 48-hour trajectory in one pass
Architecture Insight: The model uses attention pooling to create a context vector from the encoded sequence, then conditions forecasting on both this context and the most recent observations for better short-term accuracy.
